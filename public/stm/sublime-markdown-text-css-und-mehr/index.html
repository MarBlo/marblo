Sublime

Dictionary

25Jan2020

Mit der Anleitung aus diesm Link habe ich ein deutsches Wörterbuch eingebaut. Ein etwas länglicher aber einfacher Vorgang. Dabei hat aber der Vorgang mit dem zip en der Datei nicht so richtig funktioniert. Was dann funktioniert hat war, die Einrichtung eines Ordners über Preferences –> Browse Packages und dann in diesm Ordner einen Ordner mit dem Namen Language-German erstellen und in diesen die beiden .dic und .aff files legen.

Über F6 kann man dann das Dictionary aktivieren.

Noch zeigt das Dictionary aber auch alle Wörter mit Umlaut als Fehler an.

Dafür macht es aber auch Vorschläge, die wohl aus dem Dictionary kommen.

Snippets

26Jan2020

In diesem Link freeCodeCamp gibt es eine gute Erklärung zur Erstellung von Snippets an die ich mich gehalten haben. So habe ich zum Beispiel über das Kürzel mkb ein Snippet erstellt, um einen etwas größeren Text (1.2 em) zu erstellen. Dieses Snippet funktioniert genauso wie das in VS Code, allerdings mit einem anderen Shortcut.

Wichtig Bei der Erstellung des Snippets, muss man nur die Endung sublime-snippet als Filebezeichnung eingeben und abspeichern. Das Snippet wird dann automatisch in ~/Library/Application Support/Sublime Text 3/Packages/User abgelegt.

Erstellen eines neuen Snippets über Tools –> Developer –> New Snippet .
